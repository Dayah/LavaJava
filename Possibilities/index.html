Below is a list of every HTML tag we can use in your Web
Page. Some tags may not be viewable in all browsers.

Tag Description
A Makes a Link
ABBREV Sets Apart an Abbreviation
ACRONYM Sets Apart an Acronym
ADDRESS Makes Text Italic
AREA Makes a Hot Spot in an Image Map
AU Denotes the Document's Author
B Makes Text Bold
BANNER Denotes a Non-Scrolling Area
BASE Specifies a Document's URL
BASEFONT Sets Default Font Size
BGSOUND Adds a Background Sound
BIG Makes Text Big
BLINK Makes Text Blink (Very Annoying)
BLOCKQUOTE Sets Apart a Quotation
BODY Where the Document's Body is
BR Inserts a Line Break
CAPTION Makes a Caption for a Table
CENTER Centers Text and Images
CITE Makes Text Italic
CODE Specifies a Code Sample
COMMENT Indicates a Comment
DEL Denotes Deleted Text
DD Definition in a Definition List
DFN Makes Text Italic
DIR Denotes a Directory List
DIV Denotes a Division (Like P)
DL Denotes a Directory List
DT Specifies a Definition Term
EM Makes Text Italic
FIG Like IMG but More Enhanced
FONT Changes the Font
FORM Denotes a Form
FRAME Separates Window into Frames
FRAMESET The Main Part of a Frame
Hn Makes Text in a Heading Style
HR Makes a Horizontal Rule
HTML Denotes an HTML Document
I Makes Text Italic
IMG Denotes an Image
INPUT Specifies a Form Input Field
INS Denotes Inserted Text
ISINDEX Makes a Document Searchable
KBD Makes Text Bold and Fixed-Width
LANG Denotes the Document's Language
LH Makes an Expandible List
LI Denotes One List Item
LISTING Makes Text Fixed-Width
MAP Many Hot Spots on an Image Map
MARQUEE Makes a Scrolling Text Marquee
MATH Inserts a Math Expression
MENU Denotes a List of Items
META Different Client-Pull Features
NOBR Turns off Line Breaking
NOFRAMES Viewable by Non-Frames Browsers
OBJECT Embeds an ActiveX Control
OL Denotes an Ordered List
OPTION Denotes a Choice in a List Box
P Denotes a Paragraph
PERSON Sets off a Person's Name
PLAINTEXT Makes Text Fixed-Width Without Tags
PRE Makes Text Fixed Width
Q Denotes a Quote
S Makes Text Striked Through
SAMP Specifies a Code Sample
SELECT Starts a List Box or Dropdown List
SMALL Makes Text Small
STRIKE Makes Text Striked Through
STRONG Makes Text Bold
SUB Makes Text Subscript
SUP Makes Text Superscript
TAB Tabs Text to a Certain Location
TABLE Inserts a Table
TITLE Gives the Document a Title
TT Makes Text Fixed-Width
U Makes Text Underlined
UL Denotes an Unordered List
VAR Makes Text Small and Fixed-Width
WBR Inserts a Soft Linebreak in NOBR
XMP Makes Text Fixed-Width
